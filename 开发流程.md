# STM32F103C8T6 编码器接口测速 — 开发流程

## 1. 项目概述

- **MCU**: STM32F103C8T6（Cortex-M3, 64KB Flash, 20KB SRAM, 72MHz）
- **外设**: TIM3 编码器接口 + TIM2 定时中断 + OLED 显示
- **通信方式**: 编码器接口（PA6/PA7, TI12 双边沿）+ 软件 I2C（OLED）
- **构建工具**: Keil MDK uVision5 / EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)
- **代码仓库**: [GitHub: rjsjing/stm32-encoder-speed](https://github.com/rjsjing/stm32-encoder-speed)

---

## 2. 技术栈

- **MCU**: STM32F103C8T6（Cortex-M3）
- **库**: ST 标准外设库 v3.5.0
- **构建工具**: Keil MDK uVision5 + EIDE (VSCode 插件 `cl.eide`)
- **调试器**: ST-Link (SWD)

---

## 3. 硬件连接

| STM32F103C8T6 | 连接目标 |
|---------------|----------|
| PA6           | 编码器 A 相（TIM3_CH1, IPU） |
| PA7           | 编码器 B 相（TIM3_CH2, IPU） |
| PB5           | OLED SCL |
| PB6           | OLED SDA |
| 3.3V          | VCC     |
| GND           | GND     |

## 5. 开发步骤

### 5.1 依赖

- Keil MDK v5 + STM32F10x 设备包
- VSCode + EIDE 插件 (`cl.eide`)
- 编译宏: `USE_STDPERIPH_DRIVER`, `STM32F10X_MD`

### 5.2 编译

```
Keil:  打开 Project.uvprojx → F7 编译
EIDE:  Ctrl+Shift+P → "EIDE: Build"
```

### 5.3 烧录

```
Keil:  F8 烧录（ST-Link SWD）
CLI:   STM32_Programmer_CLI.exe -c port=SWD -w build/Target\ 1/Project.hex
```

### 5.4 功能验证

- 旋转编码器，OLED 实时显示速度值
- TIM3 编码器模式自动处理 A/B 相方向和计数
